Detailed Description

The SimManager (or simulation manager) is used to coordinate the activities and data of two different simulations in order to connect their functionality and offer support for front-end applications.

Initializes the simulation manager using the given file pathname and types of Reality and Knowledge sims.

To use custom Reality or Knowledge Sim implementations, this method should be called and modified in any user-facing or client programs

Example usage: SimManager.Init("myPath.json", typeof(MyRealitySim), typeof(MyKnowledgeSim)

Parameters
