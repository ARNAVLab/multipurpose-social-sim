static bool WorldManager.RegisterAgent ( Actor registree)
static

Attempts to set which Agent is "focused" (is selected/hovered on the frontend).

This fails if there is currently a focused Agent.

Parameters
targetIDis the ID of the Agent to focus.
Returns
whether the currently focused Agent successfully changed. Attempts to unfocus the currently "focused" (is selected/hovered on the frontend) Agent. This fails if the currently focused Agent's ID does not match the passed ID.
Parameters
targetIDis the ID of the Agent to unfocus.
Returns
whether the currently focused Agent was successfully unfocused. Attempts to add a new Agent to the static Dictionary, using its AgentID as the key. This fails if the Dictionary already contains a value with the desired key.
Parameters
registreeis the Agent to register.
Returns
whether the addition was successful or not.
